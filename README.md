# Volvo-Cars-QA-Portfolio
QA Analysis and Test Case Design for Volvocars.com - Focus on Car Configurator and User Journey.
## 📌 Project Overview
A professional QA study of **volvocars.com** to demonstrate my ability to analyze complex systems, design data-driven test cases, and identify UX improvements.

## 🎯 Scope of Testing
* **Functional Logic:** Pricing updates and accessory dependencies.
* **Mobile & Web:** UI/UX consistency across different viewports.
* **User Journeys:** End-to-end flow from car selection to checkout/booking.

## 📁 Repository Structure
* **/Test-Cases**: Structured test suites in Google Sheets (Functional & Edge Cases).
* **/Observations**: Documenting potential UX improvements and system behaviors.
* **/Evidence**: Visual documentation of the testing process.

## 🛠 Skills & Tools
* **Methods:** ISTQB-based Testing, BVA, Equivalence Partitioning.
* **Tools:** Chrome DevTools, uTest, Google Sheets, Mobile Emulators.
